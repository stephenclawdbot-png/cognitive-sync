## Creating Demo Video

To record a demo:

1. Start the application: `docker-compose up`
2. Open browser to http://localhost:8000
3. Use screen recording software:
   - **macOS**: Cmd+Shift+5 for QuickTime
   - **Windows**: Xbox Game Bar (Win+G)
   - **Linux**: SimpleScreenRecorder

4. Record 30-60 seconds showing:
   - Initial 3D space with thoughts
   - Real-time thought generation
   - User movement
   - State transitions

5. Save as `demo-v1.1.mp4` in this directory
